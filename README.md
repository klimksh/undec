### How do I get set up? ###

* set up virtualenv (see documentation), then
	
    ```
		virtualenv env
	```
	
* activate environment
	```
		source env/bin/activate
	```
	
* install requirements
	```
    pip install -r req.txt
	```
	
* run python notebook
    ```
    jupyter notebook
    ```
